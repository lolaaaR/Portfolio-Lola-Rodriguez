NovaHub Tienda de Juegos

E-commerce en ASP.NET WebForms con carrito, roles y validación de stock

1 / 6

¿Qué hace?

NovaHub es una tienda de videojuegos online desarrollada en equipo para la facultad con ASP.NET WebForms. Permite a los usuarios buscar y comprar productos físicos y digitales, gestionar un carrito de compras dinámico, y diferencia la experiencia según el rol del usuario logueado (cliente o administrador).

Funcionalidades principales

  • Buscador en la barra de navegación que filtra el catálogo por texto
  • Carrito de compras con contador de artículos actualizado en tiempo real
  • Botones de sumar/restar cantidad directamente desde el listado de productos
  • Validación de stock, diferenciando productos físicos de digitales
  • Vistas y permisos distintos según el rol del usuario (cliente / vendedor / administrador)
  • Edición de productos habilitada solo para usuarios con rol de administrador
  • Persistencia de datos con base de datos SQL Server (productos, usuarios, pedidos)

Mi rol en el equipo

Dentro del equipo me enfoqué en la lógica de roles de usuario, implementando los permisos y vistas diferenciadas para administrador y vendedor. También trabajé en algunos filtros de búsqueda del catálogo, y en el desarrollo del inicio de sesión y el registro de nuevos usuarios.

Tecnologías

C# ASP.NET WebForms SQL Server HTML CSS Git & GitHub
