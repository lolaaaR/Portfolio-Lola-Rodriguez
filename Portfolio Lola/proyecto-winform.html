App de Escritorio (WinForm)

Sistema de gestión de artículos con C# y SQL Server

1 / 6

¿Qué hace?

Es una aplicación de escritorio desarrollada en equipo para la facultad, que permite gestionar el inventario de una tienda: alta, baja y modificación de artículos, categorías y marcas, con carga de imágenes y validaciones de datos. Toda la información se persiste en una base SQL Server.

Funcionalidades principales

  • Login y validación de usuario
  • ABM completo de artículos, categorías y marcas
  • Carga y visualización de imágenes por artículo
  • Validaciones de formulario antes de guardar
  • Conexión y consultas a SQL Server

Tecnologías

C# WinForm SQL Server Git & GitHub
