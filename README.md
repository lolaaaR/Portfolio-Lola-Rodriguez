❤️ Mi Portfolio  - Lola Rodriguez

Este proyecto es mi portfolio personal, diseñado para mostrar mi camino como Desarrolladora Full Stack y los proyectos en los que voy a ir trabajando.

✨ Características Principales
Diseño Moderno: Estética con transparencias y desenfoques.

Totalmente Responsivo: Se adapta perfectamente a celulares, tablets y pc.

Navegación Fluida: Desplazamiento suave (smooth scroll) entre secciones.

Interactividad: Carruseles de imágenes para proyectos y hover con efectos especiales.

- Herramientas utilizadas
HTML: Estructura

CSS: Diseño, Flexbox, Grids y animaciones.

JavaScript: Lógica para carruseles.
